# Business Detail Modal

## Summary
A modal dialog that displays comprehensive details about a selected business, including photos, reviews, hours, and contact info.

## Details
- Opened by clicking "View" on a results table row
- Displays a photo carousel from Google Places photos
- Shows all reviews with author attribution and star ratings
- Lists opening hours by day of week
- Provides copy-to-clipboard for phone and address
- Links to Google Maps listing

## Key Files
- `app.js` — `openDetailModal()`, `getPhotoUrl()`
- `styles.css` — Modal overlay and content styling
- `index.html` — Modal container element

## Features
- Photo gallery with navigation
- Review cards with author names and ratings
- Operating hours display
- Copy-to-clipboard for contact info
- Responsive layout for mobile
